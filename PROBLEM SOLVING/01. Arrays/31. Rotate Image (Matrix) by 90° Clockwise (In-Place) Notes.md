# PROBLEM (Original Statement)

**Rotate Image**

You are given an **n × n 2D matrix representing an image**.

Rotate the image **90 degrees clockwise**.

You must rotate the matrix **in-place**, meaning you **cannot use another matrix for storage**.

Example:

Input

```
matrix =
1 2 3
4 5 6
7 8 9
```

Output

```
7 4 1
8 5 2
9 6 3
```

---

# Topics / Patterns

### Topic

Matrix Manipulation

### Patterns

1. **Matrix Transformation**
2. **Transpose + Reverse Trick**

Why?

Because a **90° clockwise rotation** can be mathematically achieved by:

```
Transpose matrix
+
Reverse each row
```

This pattern appears frequently in **matrix transformation problems**.

# BRUTE FORCE APPROACH

## First Thought Intuition

First natural idea:

When we rotate a matrix, elements move to **new positions**.

Example:

```
matrix[i][j]
```

moves to

```
matrix[j][n - i - 1]
```

So simplest thought:

```
Create new matrix
Place elements in rotated position
```

Then copy back.

This is the most natural beginner idea.

---

## Thought Process (Step-wise)

Step 1
Create a new matrix `rotated`.

Step 2
Traverse original matrix.

Step 3
For each element:

```
rotated[j][n-i-1] = matrix[i][j]
```

Step 4
Copy rotated matrix back.

---

## Pseudocode

```
create new matrix rotated[n][n]

for i from 0 to n-1
   for j from 0 to n-1
       rotated[j][n-i-1] = matrix[i][j]

copy rotated into original matrix
```

---

## Algorithm

1. Create temporary matrix
2. Traverse original matrix
3. Place elements in rotated position
4. Copy result back

---

# CODE

## C++

### V1 (Only Required Function)

```cpp
void rotate(vector<vector<int>>& matrix) {

    int n = matrix.size();

    vector<vector<int>> temp(n, vector<int>(n));

    for(int i = 0; i < n; i++) {
        for(int j = 0; j < n; j++) {

            temp[j][n - i - 1] = matrix[i][j];

        }
    }

    matrix = temp;
}
```

---

### V2 (Complete Program)

```cpp
#include <iostream>
#include <vector>
using namespace std;

void rotateMatrix(vector<vector<int>>& matrix)
{
    int n = matrix.size();

    // Temporary matrix to store rotated values
    vector<vector<int>> temp(n, vector<int>(n));

    // Traverse original matrix
    for(int i = 0; i < n; i++)
    {
        for(int j = 0; j < n; j++)
        {
            // Place element at rotated position
            temp[j][n - i - 1] = matrix[i][j];
        }
    }

    // Copy back to original matrix
    matrix = temp;
}

int main()
{
    vector<vector<int>> matrix =
    {
        {1,2,3},
        {4,5,6},
        {7,8,9}
    };

    rotateMatrix(matrix);

    // Print matrix
    for(auto &row : matrix)
    {
        for(auto &val : row)
        {
            cout << val << " ";
        }
        cout << endl;
    }

    return 0;
}
```

# BETTER APPROACH

## First Thought Intuition

Brute force worked but used **extra matrix**.

We must think:

> Can we rotate the matrix **without creating another matrix**?

Let’s observe how elements move.

Original

```
1 2 3
4 5 6
7 8 9
```

Rotated

```
7 4 1
8 5 2
9 6 3
```

Look at **corners movement**

```
1 → top right
3 → bottom right
9 → bottom left
7 → top left
```

This means we are basically **rotating elements layer by layer**.

So we can rotate **4 elements at a time**.

Example:

```
top → right
right → bottom
bottom → left
left → top
```

So we swap 4 values in a cycle.

This allows rotation **without extra space**.

---

## Thought Process (Step-wise)

Matrix rotates **layer by layer**.

For a 3×3 matrix:

```
Layer 1
(outer square)

1 2 3
4   6
7 8 9
```

Rotate four elements:

```
matrix[i][j]
matrix[j][n-1-i]
matrix[n-1-i][n-1-j]
matrix[n-1-j][i]
```

Perform **4-way swap**.

---

## Pseudocode

```
n = matrix size

for each layer
    for each element in layer

        temp = matrix[i][j]

        matrix[i][j] = matrix[n-1-j][i]
        matrix[n-1-j][i] = matrix[n-1-i][n-1-j]
        matrix[n-1-i][n-1-j] = matrix[j][n-1-i]
        matrix[j][n-1-i] = temp
```

---

## Algorithm

1. Traverse layers from outer to inner
2. Rotate elements in groups of four
3. Use one temporary variable
4. Complete rotation

---

# CODE

## C++ V1 (Only Required Function)

```cpp
void rotate(vector<vector<int>>& matrix) {

    int n = matrix.size();

    for(int i = 0; i < n/2; i++) {

        for(int j = i; j < n-i-1; j++) {

            int temp = matrix[i][j];

            matrix[i][j] = matrix[n-1-j][i];
            matrix[n-1-j][i] = matrix[n-1-i][n-1-j];
            matrix[n-1-i][n-1-j] = matrix[j][n-1-i];
            matrix[j][n-1-i] = temp;

        }
    }
}
```

---

## C++ V2 (Complete Program)

```cpp
#include <iostream>
#include <vector>
using namespace std;

void rotateMatrix(vector<vector<int>>& matrix)
{
    int n = matrix.size();

    // Traverse each layer
    for(int i = 0; i < n/2; i++)
    {
        for(int j = i; j < n-i-1; j++)
        {
            // Store first value
            int temp = matrix[i][j];

            // Move left → top
            matrix[i][j] = matrix[n-1-j][i];

            // Move bottom → left
            matrix[n-1-j][i] = matrix[n-1-i][n-1-j];

            // Move right → bottom
            matrix[n-1-i][n-1-j] = matrix[j][n-1-i];

            // Move temp → right
            matrix[j][n-1-i] = temp;
        }
    }
}

int main()
{
    vector<vector<int>> matrix =
    {
        {1,2,3},
        {4,5,6},
        {7,8,9}
    };

    rotateMatrix(matrix);

    for(auto row : matrix)
    {
        for(auto val : row)
            cout << val << " ";

        cout << endl;
    }
}
```

# OPTIMAL APPROACH

## First Thought Intuition

Observe something interesting.

Original

```
1 2 3
4 5 6
7 8 9
```

Step 1 → **Transpose Matrix**

Swap across diagonal.

```
1 4 7
2 5 8
3 6 9
```

Step 2 → **Reverse each row**

```
7 4 1
8 5 2
9 6 3
```

Which is exactly the rotated matrix.

So rotation =

```
Transpose + Reverse Row
```

This is the **most elegant solution**.

---

## Thought Process (Step-wise)

Step 1
Transpose matrix

```
matrix[i][j] ↔ matrix[j][i]
```

Step 2
Reverse each row.

---

## Pseudocode

```
n = matrix size

for i = 0 to n
   for j = i+1 to n
       swap(matrix[i][j], matrix[j][i])

for each row
   reverse(row)
```

---

## Algorithm

1. Transpose matrix
2. Reverse each row
3. Done

---

# CODE

## C++ V1

```cpp
void rotate(vector<vector<int>>& matrix) {

    int n = matrix.size();

    for(int i = 0; i < n; i++)
    {
        for(int j = i+1; j < n; j++)
        {
            swap(matrix[i][j], matrix[j][i]);
        }
    }

    for(int i = 0; i < n; i++)
    {
        reverse(matrix[i].begin(), matrix[i].end());
    }
}
```

## C++ V2 (Complete Program)

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

void rotateMatrix(vector<vector<int>>& matrix)
{
    int n = matrix.size();

    // Step 1: Transpose matrix
    for(int i = 0; i < n; i++)
    {
        for(int j = i+1; j < n; j++)
        {
            swap(matrix[i][j], matrix[j][i]);
        }
    }

    // Step 2: Reverse every row
    for(int i = 0; i < n; i++)
    {
        reverse(matrix[i].begin(), matrix[i].end());
    }
}

int main()
{
    vector<vector<int>> matrix =
    {
        {1,2,3},
        {4,5,6},
        {7,8,9}
    };

    rotateMatrix(matrix);

    for(auto row : matrix)
    {
        for(auto val : row)
            cout << val << " ";

        cout << endl;
    }
}
```
