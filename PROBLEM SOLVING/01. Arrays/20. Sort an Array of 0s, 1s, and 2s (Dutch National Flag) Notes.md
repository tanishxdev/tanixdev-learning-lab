# PROBLEM (Original Statement)

Given an array `nums` containing **0, 1, and 2**, sort the array **in-place** so that:

```text
0 → red
1 → white
2 → blue
```

The sorted order must be:

```text
[0,0,...,1,1,...,2,2]
```

You **cannot use built-in sort**.

This problem is also known as:

```text
Dutch National Flag Problem
```

# BREAKDOWN PROBLEM

Input

```text
nums = [2,0,2,1,1,0]
```

Output

```text
[0,0,1,1,2,2]
```

We must **rearrange the array in-place**.

Important observation:

```text
Only 3 values exist → 0,1,2
```

Typical constraints

```

```

Implications

| Approach   | Time          |
| ---------- | ------------- |

# BETTER APPROACH

## First Thought Intuition

Since values are **only 0,1,2**, we can **count them**.

Example

```
nums = [2,0,2,1,1,0]
```

Count

```
0 → 2
1 → 2
2 → 2
```

Rebuild array.

---

## Thought Process

1. Count number of `0`
2. Count number of `1`
3. Count number of `2`
4. Fill array again.

---

## Pseudocode

```
count0 = count1 = count2 = 0

for num in nums
    update counts

fill 0s
fill 1s
fill 2s
```

---

## CODE

### C++ V1

```cpp
void sortColors(vector<int>& nums)
{
    int count0 = 0;
    int count1 = 0;
    int count2 = 0;

    for(int x : nums)
    {
        if(x == 0) count0++;
        else if(x == 1) count1++;
        else count2++;
    }

    int index = 0;

    while(count0--) nums[index++] = 0;
    while(count1--) nums[index++] = 1;
    while(count2--) nums[index++] = 2;
}
```

# OPTIMAL APPROACH

Pattern

```
Dutch National Flag Algorithm
```

---

# First Thought Intuition

Divide array into **three regions**:

```
0 region | 1 region | unknown | 2 region
```

We maintain three pointers:

```
low
mid
high
```

---

# Pointer Meaning

```
low  → position of next 0
mid  → current element
high → position of next 2
```

---

# Algorithm

While

```
mid <= high
```

Check

### Case 1

```
nums[mid] == 0
```

Swap with `low`

```
low++
mid++
```

---

### Case 2

```
nums[mid] == 1
```

Just move

```
mid++
```

---

### Case 3

```
nums[mid] == 2
```

Swap with `high`

```
high--
```

Don't move mid.

---

# Pseudocode

```
low = 0
mid = 0
high = n-1

while mid <= high

    if nums[mid] == 0
        swap(low, mid)
        low++
        mid++

    else if nums[mid] == 1
        mid++

    else
        swap(mid, high)
        high--
```

---

# CODE

### C++ V1

```cpp
void sortColors(vector<int>& nums)
{
    int low = 0;
    int mid = 0;
    int high = nums.size() - 1;

    while(mid <= high)
    {
        if(nums[mid] == 0)
        {
            swap(nums[low], nums[mid]);
            low++;
            mid++;
        }
        else if(nums[mid] == 1)
        {
            mid++;
        }
        else
        {
            swap(nums[mid], nums[high]);
            high--;
        }
    }
}
```
