# PROBLEM (Original Statement)

Given an integer array `nums`, return all the triplets `[nums[i], nums[j], nums[k]]` such that:

- `i != j`
- `i != k`
- `j != k`
- `nums[i] + nums[j] + nums[k] == 0`

The solution set must not contain duplicate triplets.

---

## Topics / Patterns

**Topic:** Arrays

**Patterns:**

1. Sorting
   - Helps handle duplicates.
   - Enables two-pointer approach.

2. Two Pointer (Nested Fix-One Pattern)
   - Fix one element.
   - Use two-pointer for remaining two elements.

3. Duplicate Skipping Pattern
   - Essential to avoid repeated triplets.

This is a classic upgrade of:

- Two Sum → 3Sum

---

## BREAKDOWN PROBLEM

We are given an array.

We must:

- Find all unique triplets whose sum is 0.
- No duplicate triplets allowed.

Important:

Output is list of lists.
Order inside triplet does not matter.
Order of result does not matter.

---

## CONSTRAINT UNDERSTANDING

Typical constraints:

- 3 ≤ nums.length ≤ 3000
- -10⁵ ≤ nums[i] ≤ 10⁵

Important implications:

If we try all triplets:

- O(n³) → 3000³ = 27 billion → impossible.

So we must reduce complexity.

Target time complexity: O(n²)

---

# BRUTE FORCE APPROACH

## First Thought Intuition

Natural thought:

Try every possible triplet.

Because:

- Problem directly says choose 3 elements.

---

## Thought Process (Step-wise)

1. Loop i from 0 to n-1
2. Loop j from i+1 to n-1
3. Loop k from j+1 to n-1
4. If sum == 0 → store triplet
5. Use set to avoid duplicates

---

## Pseudocode

```
create empty set result

for i in range n:
    for j in range i+1:
        for k in range j+1:
            if nums[i] + nums[j] + nums[k] == 0:
                sort triplet
                insert into set

return set as list
```

---

## Algorithm

1. Triple nested loops.
2. Check sum.
3. Sort triplet.
4. Store in set to remove duplicates.

---

## CODE

### C++

#### V1 (Only Required Function)

```cpp
vector<vector<int>> threeSum(vector<int>& nums) {

    int n = nums.size();
    set<vector<int>> uniqueTriplets;  // To remove duplicates

    for (int i = 0; i < n; i++) {

        for (int j = i + 1; j < n; j++) {

            for (int k = j + 1; k < n; k++) {

                if (nums[i] + nums[j] + nums[k] == 0) {

                    vector<int> triplet = {nums[i], nums[j], nums[k]};
                    sort(triplet.begin(), triplet.end());

                    uniqueTriplets.insert(triplet);
                }
            }
        }
    }

    return vector<vector<int>>(uniqueTriplets.begin(), uniqueTriplets.end());
}
```

# BETTER APPROACH

## First Thought Intuition

Instead of picking 3 numbers:

Fix one number.
Then solve Two Sum for remaining.

This reduces one loop.

---

## Thought Process (Step-wise)

1. Sort array.
2. For each i:
   - target = -nums[i]
   - Use hash set to find two sum in remaining part.

---

## Pseudocode

```
sort nums

for i from 0 to n:
    create empty set seen
    for j from i+1 to n:
        complement = -nums[i] - nums[j]
        if complement in seen:
            add triplet
        insert nums[j] into seen
```

---

## Complexity

Time: O(n²)
Space: O(n)

Better than brute.

But still uses extra set and duplicate handling complex.

---

## Does This Approach Fail?

It works.

But:

- Duplicate handling becomes messy.
- Space O(n)

We can do even better and cleaner.

# OPTIMAL APPROACH

## First Thought Intuition

Sort array.

Fix one element.

Use two pointers:

- Left
- Right

Classic:

3Sum = Fix + Two Pointer

---

## Thought Process (Step-wise)

1. Sort array.
2. Loop i from 0 to n-3
3. Skip duplicate i values.
4. Set left = i+1
5. Set right = n-1
6. While left < right:
   - sum = nums[i] + nums[left] + nums[right]
   - if sum == 0:
     - store triplet
     - skip duplicates for left & right

   - if sum < 0:
     left++
   - else:
     right--

Why sorting helps?

- Enables moving left/right intelligently.
- Makes duplicate skipping easy.

---

## Pseudocode

```
sort nums

for i from 0 to n-3:

    if i > 0 and nums[i] == nums[i-1]:
        continue

    left = i+1
    right = n-1

    while left < right:

        sum = nums[i] + nums[left] + nums[right]

        if sum == 0:
            add triplet
            skip duplicates left
            skip duplicates right
            left++
            right--

        else if sum < 0:
            left++

        else:
            right--
```

---

## CODE

### C++

#### V1 (Only Required Function)

```cpp
vector<vector<int>> threeSum(vector<int>& nums) {

    vector<vector<int>> result;

    sort(nums.begin(), nums.end());  // Step 1: Sort array

    int n = nums.size();

    for (int i = 0; i < n - 2; i++) {

        // Skip duplicate values for i
        if (i > 0 && nums[i] == nums[i - 1])
            continue;

        int left = i + 1;
        int right = n - 1;

        while (left < right) {

            int sum = nums[i] + nums[left] + nums[right];

            if (sum == 0) {

                result.push_back({nums[i], nums[left], nums[right]});

                // Skip duplicate left
                while (left < right && nums[left] == nums[left + 1])
                    left++;

                // Skip duplicate right
                while (left < right && nums[right] == nums[right - 1])
                    right--;

                left++;
                right--;
            }

            else if (sum < 0)
                left++;   // Need bigger value

            else
                right--;  // Need smaller value
        }
    }

    return result;
}
```

#### V2 (Complete Program)

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

vector<vector<int>> threeSum(vector<int>& nums) {

    vector<vector<int>> result;

    sort(nums.begin(), nums.end());  // Sorting helps two pointer

    int n = nums.size();

    for (int i = 0; i < n - 2; i++) {

        // Skip duplicate starting elements
        if (i > 0 && nums[i] == nums[i - 1])
            continue;

        int left = i + 1;
        int right = n - 1;

        while (left < right) {

            int sum = nums[i] + nums[left] + nums[right];

            if (sum == 0) {

                result.push_back({nums[i], nums[left], nums[right]});

                // Skip duplicate values at left pointer
                while (left < right && nums[left] == nums[left + 1])
                    left++;

                // Skip duplicate values at right pointer
                while (left < right && nums[right] == nums[right - 1])
                    right--;

                left++;
                right--;
            }
            else if (sum < 0) {
                left++;   // Need bigger number
            }
            else {
                right--;  // Need smaller number
            }
        }
    }

    return result;
}

int main() {

    vector<int> nums = {-1, 0, 1, 2, -1, -4};

    vector<vector<int>> result = threeSum(nums);

    for (auto triplet : result) {
        for (int num : triplet) {
            cout << num << " ";
        }
        cout << endl;
    }

    return 0;
}
```
