* Complex state handling
* Spaghetti code
* Slow performance
* UI bugs due to sync issues

## 2. What is React?

React is:

> A JavaScript library for building fast, scalable, component-based user interfaces using a declarative approach.

Key ideas:

* UI = function of state
* Component-based architecture
* Automatic DOM updates
* Virtual DOM for performance

React only focuses on:
âœ… View Layer (UI)
