## 1. Problem Without Proper Styling System

In plain HTML you write:

```html
<div style="color:red; font-size:20px">
```

In large React apps this causes:

* Inline style repetition
* No structure
* Hard to manage design
* Global CSS conflicts
* Poor modularity

We need **structured, scalable styling approaches**.

---

## 2. Core Concept

React supports multiple styling methods:

1. Inline Styles
2. External CSS Files
3. CSS Modules
4. Dynamic Styles with JavaScript

We will implement the two most important for fundamentals:
✅ Inline + External CSS

## 4. Code Implementation

### A. External CSS file

**src/styles/StyledBox.css**

```css
.box {
  background-color: lightblue;
  padding: 20px;
  border-radius: 5px;
  margin-top: 10px;
}
```

---

### B. Component Using Styles

**src/components/StyledBox.jsx**

```jsx
// StyledBox.jsx
// Demonstrates multiple styling methods in React.

import "../styles/StyledBox.css";

function StyledBox() {

  // Inline style object
  const inlineStyle = {
    color: "white",
    backgroundColor: "green",
    padding: "10px"
  };

  return (
    <div>
      <h2>React Styling Example</h2>

      {/* External CSS */}
      <div className="box">
        This uses External CSS Styling
      </div>

      {/* Inline Styling */}
      <div style={inlineStyle}>
        This uses Inline Styling
      </div>
    </div>
  );
}

export default StyledBox;
```

---

### C. Using in App.jsx

**src/App.jsx**

```jsx
import StyledBox from "./components/StyledBox";

function App() {
  return (
    <div>
      <h1>Styling Concept</h1>
      <StyledBox />
    </div>
  );
}

export default App;
```

## 5. How Styling Works in React

### External CSS

* Works like normal CSS
* Uses className
* Scoped globally

### Inline Styles

* Passed as JavaScript object
* camelCase properties
* Dynamic possible

## 8. When To Use What

| Method            | Use Case                 |
| ----------------- | ------------------------ |
| External CSS      | Large consistent styling |
| Inline            | Dynamic, quick styling   |
| CSS Modules       | Component-scoped styling |
| Styled-components | Advanced theming         |
