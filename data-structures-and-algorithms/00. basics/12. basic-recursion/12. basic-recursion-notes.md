# BASIC RECURSION

## What is Recursion?

Recursion is a technique where a function calls itself to solve smaller versions of the same problem.

Every recursive function must have:

1. Base Case → where recursion stops
2. Recursive Call → where function calls itself

If base case is missing → infinite recursion → stack overflow.

---

# GENERAL TEMPLATE

```cpp
void func(int n)
{
    if(n == 0) return;            // base case
    func(n - 1);                  // recursive call
}
```

## 1. Understand Recursion by Printing Something N Times

### Problem

Print "Hello" N times using recursion.

### Logic

Decrease N at each call.

### Code

```cpp
void printHello(int n)
{
    if(n == 0) return;

    cout << "Hello" << endl;
    printHello(n - 1);
}
```

### Example

Input: n = 3
Output:

```
Hello
Hello
Hello
```

## 3. Print 1 to N Using Recursion

### Concept

We first go to base, then print while returning.

### Code

```cpp
void print1toN(int n)
{
    if(n == 0) return;

    print1toN(n - 1);
    cout << n << " ";
}
```

### Example

n = 5
Output:


## 5. Sum of First N Numbers

### Formula

sum(n) = n + sum(n-1)

### Code

```cpp
int sumN(int n)
{
    if(n == 0) return 0;
    return n + sumN(n - 1);
}
```

### Example

n = 5
5 + 4 + 3 + 2 + 1 = 15

---

## 6. Factorial of N Using Recursion

### Formula

n! = n × (n-1)!

### Code

```cpp
int factorial(int n)
{
    if(n == 0 || n == 1) return 1;
    return n * factorial(n - 1);
}
```

### Example

5! = 120

---

## 7. Reverse an Array Using Recursion

### Logic

Swap start and end, then move inward.

### Code

```cpp
void reverseArray(int arr[], int start, int end)
{
    if(start >= end) return;

    swap(arr[start], arr[end]);
    reverseArray(arr, start + 1, end - 1);
}
```

### Example

Input:

```
1 2 3 4 5
```

Output:

```
5 4 3 2 1
```

---

## 8. Check if String is Palindrome Using Recursion

### Logic

Compare first and last characters recursively.

### Code

```cpp
bool isPalindromeRec(string &s, int l, int r)
{
    if(l >= r) return true;
    if(s[l] != s[r]) return false;

    return isPalindromeRec(s, l + 1, r - 1);
}
```

### Example

Input: "madam" → TRUE
Input: "hello" → FALSE

---

## 9. Fibonacci Number using Recursion

### Formula

fib(n) = fib(n-1) + fib(n-2)

### Code

```cpp
int fibonacci(int n)
{
    if(n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}
```

### Example

fib(6) = 8
Sequence: 0 1 1 2 3 5 8
