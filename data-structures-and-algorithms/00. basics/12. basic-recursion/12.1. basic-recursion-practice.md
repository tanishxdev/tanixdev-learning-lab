## Q1 Solution: Print N to 0

### Thinking (Before Code)

* We need to print first, then go smaller
* Base case: when `n < 0`, stop
* Print happens **before recursive call**

### Code (With Comments)

```cpp
void printNtoZero(int n)
{
    if(n < 0) return;        // base case

    cout << n << " ";        // work
    printNtoZero(n - 1);     // recursive call
}
```

### Dry Run

```
n=3 ‚Üí print 3
n=2 ‚Üí print 2
n=1 ‚Üí print 1
n=0 ‚Üí print 0
n=-1 ‚Üí stop
```

## Q2 Solution: Countdown + Done

### Thinking

* Numbers must print first
* `"Done"` should print **after recursion completes**
* So print `"Done"` AFTER recursive call

### Code

```cpp
void countdown(int n)
{
    if(n == 0) return;

    cout << n << endl;
    countdown(n - 1);
}
```

Usage:

```cpp
countdown(3);
cout << "Done";
```

### Dry Run

```
countdown(3)
 ‚Üí print 3
 ‚Üí countdown(2)
   ‚Üí print 2
   ‚Üí countdown(1)
     ‚Üí print 1
     ‚Üí countdown(0) stop
Back to main ‚Üí print Done
```

## Q3 Solution: Sum of Digits

### Thinking

* Last digit = n % 10
* Remaining number = n / 10
* Add last digit + recursive result

### Code

```cpp
int sumDigits(int n)
{
    if(n == 0) return 0;          // base case

    return (n % 10) + sumDigits(n / 10);
}
```

### Dry Run (n = 123)

```
sumDigits(123)
= 3 + sumDigits(12)
= 3 + 2 + sumDigits(1)
= 3 + 2 + 1 + sumDigits(0)
= 6
```

## Q4 Solution: Count Digits

### Thinking

* Each recursive call removes one digit
* Count increases by 1 each time

### Code

```cpp
int countDigits(int n)
{
    if(n == 0) return 0;

    return 1 + countDigits(n / 10);
}
```

### Dry Run (456)

```
countDigits(456)
= 1 + countDigits(45)
= 1 + 1 + countDigits(4)
= 1 + 1 + 1 + countDigits(0)
= 3
```

## Q5 Solution: Power (a^b)

### Thinking

* Base case: b == 0 ‚Üí return 1
* Recursive case: a * power(a, b-1)

### Code

```cpp
int power(int a, int b)
{
    if(b == 0) return 1;

    return a * power(a, b - 1);
}
```

### Dry Run (2^3)

```
power(2,3)
= 2 * power(2,2)
= 2 * 2 * power(2,1)
= 2 * 2 * 2 * power(2,0)
= 8
```

## Q6 Solution: Print Array Elements

### Thinking

* Use index to move forward
* Stop when index reaches size

### Code

```cpp
void printArray(int arr[], int index, int n)
{
    if(index == n) return;

    cout << arr[index] << " ";
    printArray(arr, index + 1, n);
}
```

### Dry Run

```
index=0 ‚Üí print arr[0]
index=1 ‚Üí print arr[1]
index=2 ‚Üí print arr[2]
index=3 ‚Üí stop
```

## Q7 Solution: Maximum Element (Recursion)

### Thinking

* Compare current element with max of rest
* Reduce array size each call

### Code

```cpp
int maxElement(int arr[], int n)
{
    if(n == 1) return arr[0];

    return max(arr[n-1], maxElement(arr, n-1));
}
```

### Dry Run (1 5 2)

```
max(2, max(1,5))
= max(2,5)
= 5
```

## Q8 Solution: Even Check

### Thinking

* Base case: 0 is even
* Reduce number by 2 each call

### Code

```cpp
bool isEven(int n)
{
    if(n == 0) return true;
    if(n == 1) return false;

    return isEven(n - 2);
}
```

### Dry Run (6)

```
isEven(6)
‚Üí isEven(4)
‚Üí isEven(2)
‚Üí isEven(0)
‚Üí true
```
