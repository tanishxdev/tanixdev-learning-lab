# SOLUTIONS (WITH THINKING PROCESS)

## Q1 Solution: Sum of Digits

### Thinking

* Extract digits using `% 10`
* Add to sum
* Remove digit using `/ 10`

```cpp
int sumDigits(int n)
{
    int sum = 0;

    while(n > 0)
    {
        sum += n % 10;
        n /= 10;
    }
    return sum;
}
```

## Q2 Solution: Product of Digits

### Thinking

* Similar to sum
* Start product with 1

```cpp
int productDigits(int n)
{
    int prod = 1;

    while(n > 0)
    {
        prod *= n % 10;
        n /= 10;
    }
    return prod;
}
```

## Q3 Solution: Even or Odd Digit Count

### Thinking

* Count digits
* Check count % 2

```cpp
bool evenDigitCount(int n)
{
    int count = 0;

    while(n > 0)
    {
        count++;
        n /= 10;
    }

    return count % 2 == 0;
}
```

## Q6 Solution: Count Trailing Zeros

### Thinking

{

## Q7 Solution: Perfect Square

### Thinking

bool isPerfectSquare(int n)
{
    int root = sqrt(n);
    return root * root == n;

## Q8 Solution: LCM

### Thinking

* Use GCD
* Avoid overflow logic for basics

```cpp
int gcd(int a, int b)
{
    while(b != 0)
    {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int lcm(int a, int b)
{
    return (a * b) / gcd(a, b);
}
```

## Q9 Solution: Count Prime Digits

### Thinking

* Prime digits are fixed: 2,3,5,7
* Check each digit

```cpp
int countPrimeDigits(int n)
{
    int count = 0;

    while(n > 0)
    {
        int d = n % 10;
        if(d == 2 || d == 3 || d == 5 || d == 7)
            count++;
        n /= 10;
    }
    return count;
}
```

## Q10 Solution: Power of 10

### Thinking

* Power of 10 has one `1` and rest `0`s

```cpp
bool isPowerOf10(int n)
{
    if(n < 10) return false;

    while(n % 10 == 0)
        n /= 10;

    return n == 1;
}
```
