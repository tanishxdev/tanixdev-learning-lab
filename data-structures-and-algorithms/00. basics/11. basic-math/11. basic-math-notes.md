# 1. COUNT DIGITS

## Concept

Find how many digits are present in a given integer.

### Logic

Keep dividing the number by 10 until it becomes 0.
Each division removes one digit.

### Algorithm

1. count = 0
2. while(n > 0)
   n = n / 10
   count++
3. Print count

### Code

```cpp
#include <iostream>
using namespace std;

int countDigits(int n) {
    int count = 0;

    while(n > 0) {
        n = n / 10;
        count++;
    }
    return count;
}

int main() {
    int n = 12345;
    cout << countDigits(n);
}
```

### Example

n = 12345
12345 → 1234 → 123 → 12 → 1 → 0
Total steps = 5
Output = 5

# 2. REVERSE A NUMBER

## Concept

Reverse the digits of the number.

### Logic

Take last digit using % 10 and build reverse number.

### Algorithm

1. rev = 0
2. while(n > 0)
   digit = n % 10
   rev = rev * 10 + digit
   n = n / 10

### Code

```cpp
int reverseNumber(int n) {
    int rev = 0;

    while(n > 0) {
        int digit = n % 10;
        rev = rev * 10 + digit;
        n = n / 10;
    }
    return rev;
}
```

### Example

n = 1234
Step:
rev = 4 → 43 → 432 → 4321
Output = 4321

# 3. CHECK PALINDROME

## Concept

A number is palindrome if it reads same forward and backward.

### Logic

Reverse the number and compare.

### Code

```cpp
bool isPalindrome(int n) {
    int original = n;
    int rev = 0;

    while(n > 0) {
        rev = rev * 10 + (n % 10);
        n = n / 10;
    }

    return original == rev;
}
```

### Example

Input: 121
Reverse = 121
True → Palindrome

Input: 123
Reverse = 321
False

# 4. GCD / HCF (Greatest Common Divisor)

## Concept

Largest number that divides both numbers.

### Best Method – EUCLIDEAN ALGORITHM

### Logic

Keep replacing:
gcd(a, b) = gcd(b, a % b)

Until b = 0.

### Code

```cpp
int gcd(int a, int b) {
    while(b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

### Example

a = 20, b = 28
28 % 20 = 8
20 % 8 = 4
8 % 4 = 0
GCD = 4

# 5. ARMSTRONG NUMBER

## Concept

A number is Armstrong if:
Sum of each digit raised to power of number of digits = original number.

Example:
153 = 1³ + 5³ + 3³ = 153

### Logic

1. Count digits
2. For each digit: pow(digit, digits)
3. Sum them
4. Compare with original number

### Code

```cpp
#include <cmath>

bool isArmstrong(int n) {
    int original = n;
    int digits = 0;
    int temp = n;

    while(temp > 0) {
        digits++;
        temp /= 10;
    }

    int sum = 0;
    temp = n;
    while(temp > 0) {
        int digit = temp % 10;
        sum += pow(digit, digits);
        temp /= 10;
    }

    return sum == original;
}
```

### Example

153 → TRUE
370 → TRUE
123 → FALSE

# 6. PRINT ALL DIVISORS

## Concept

Divisors are numbers that divide N completely.

### Optimised Logic (O√N)

Loop till sqrt(N) and print both divisors.

### Code

```cpp
#include <vector>

void printDivisors(int n) {
    for(int i = 1; i * i <= n; i++) {
        if(n % i == 0) {
            cout << i << " ";
            if(i != n / i)
                cout << n / i << " ";
        }
    }
}
```

### Example

n = 36
Output:
1 36 2 18 3 12 4 9 6

# 7. CHECK FOR PRIME

## Concept

Prime = only divisible by 1 and itself.

### Optimised Approach

Check divisibility till √N.

### Code

```cpp
bool isPrime(int n) {
    if(n <= 1) return false;

    for(int i = 2; i * i <= n; i++) {
        if(n % i == 0)
            return false;
    }
    return true;
}
```

### Examples

7 → Prime
9 → Not Prime
1 → Not Prime
