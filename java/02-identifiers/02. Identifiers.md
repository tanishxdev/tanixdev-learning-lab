# JAVA IDENTIFIERS (FULL MASTER NOTE)

---

# 1. CONCEPT (What + Why + Mental Model)

## What is Identifier?

Identifier = **name given to programming elements**

These include:

- Variables
- Methods
- Classes
- Packages
- Interfaces

---

## Simple Definition

> Identifier = **Naam jo tum code me dete ho**

---

## Example

```java
int age = 20;
```

- `age` → Identifier (variable ka naam)

---

## Mental Model

Think like:

- Identifier = **label/tag**
- Without name → system identify nahi kar paayega

## Real Life Analogy

- Aadhaar number / Name → identify a person
- Same way → identifier identifies variable/class

# 2. SYNTAX / RULES (VERY IMPORTANT)

Agar yeh rules break kiye → **compile-time error**

---

## Rule 1: Allowed Characters

Only allowed:

- A–Z
- a–z
- 0–9
- `_` (underscore)
- `$` (dollar)

---

### Valid Examples

```
myVar
_var
$amount
sum123
```

---

### Invalid Example

```
my@var   // ❌ @ not allowed
```

---

## Rule 2: Cannot Start with Digit

```
123abc   ❌
abc123   ✅
```

---

## Rule 3: No Spaces

```
my var   ❌
myVar    ✅
```

---

## Rule 4: Case-Sensitive

```
age ≠ Age ≠ AGE
```

These are different variables

---

## Rule 5: Cannot Use Keywords

```
int class = 10;   ❌
```

Because `class` is reserved word

---

## Rule 6: Length

- No strict limit
- Recommended: **4–15 characters**

# 3. CODE (2 VERSIONS)

---

## V1 — Basic Concept Code

```java
// Demonstrating valid identifiers

int age = 21;          // valid
int _count = 10;       // valid
int $price = 100;      // valid
int totalSum = 50;     // valid
```

---

## V2 — Full Program (with explanation)

```java
// File: IdentifierDemo.java

public class IdentifierDemo {

    public static void main(String[] args) {

        // Variable declaration using valid identifiers

        int age = 20;          // normal identifier
        int _count = 5;        // underscore allowed
        int $price = 100;      // dollar allowed
        int totalSum = 50;     // camelCase (recommended)

        // Printing values
        System.out.println("Age: " + age);
        System.out.println("Count: " + _count);
        System.out.println("Price: " + $price);
        System.out.println("Total: " + totalSum);
    }
}
```

# 6. COMMON MISTAKES / INTERVIEW TRAPS

---

## Mistake 1: Using spaces

```
int my age = 10;   ❌
```

---

## Mistake 2: Starting with number

```
int 1value = 10;   ❌
```

---

## Mistake 3: Using special symbols

```
int total#sum = 50;   ❌
```

---

## Mistake 4: Using keywords

```
int while = 10;   ❌
```

---

## Mistake 5: Confusing case

```
int age = 10;
int Age = 20;   // different variable
```

# 9. BEST PRACTICES (VERY IMPORTANT)

---

## Use Naming Conventions

| Type     | Style      |
| -------- | ---------- |
| Variable | camelCase  |
| Class    | PascalCase |
| Constant | UPPER_CASE |

---

### Example

```java
int studentAge;       // good
class StudentData {}  // good
final int MAX = 100;  // good
```
