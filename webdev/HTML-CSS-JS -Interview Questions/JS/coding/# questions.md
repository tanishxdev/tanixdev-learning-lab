# JavaScript Coding Interview Questions Sheet

## **Beginner Level Questions (1-15)**

### **Coding Questions**

1. **Array Sum**: Write a function `sumArray(arr)` that returns the **sum** of all elements in an array.
2. **Palindrome Check**: Write a function `isPalindrome(str)` to check if a string is a palindrome.
3. **String Reversal**: Write a function `reverseString(str)` that reverses a given string.
4. **Maximum Difference**: Write a function `findMaxDifference(arr)` to find the maximum difference between any two elements in an array.
5. **Prime Number Check**: Write a function `isPrime(num)` that checks if a given number is prime.
6. **Remove Falsy Values**: Write a function `removeFalsyValues(arr)` that removes falsy values from an array.
7. **Fibonacci Sequence**: Write a function that returns the Fibonacci sequence up to a certain number.
8. **Array to Object**: Write a function `convertArrayToObject(arr)` that converts an array of objects into a single object.
9. **Remove Duplicates**: Write a function `removeDuplicates(arr)` that finds unique elements in an array with duplicates.
10. **Anagram Check**: Write a function `areAnagrams(str1, str2)` that checks if two strings are anagrams.
11. **Merge Arrays**: Write a function `mergeArrays(arr1, arr2)` that merges two arrays without using `concat()`.
12. **Array to String**: Write a function `convertToString(arr)` that converts an array of strings to a single string separated by spaces.
13. **Sort Descending**: Write a function `sortDescending(arr)` that takes an array of numbers and sorts it in descending order.
14. **Sum Function**: Write a function `sum(a, b)` that takes two parameters and returns their sum.
15. **Multiplication Function**: Write a function `multiply(a, b)` that takes two parameters and returns their product.

### **Theory Questions**

1. **Variable Declaration**: What is the difference between `var`, `let`, and `const`?
2. **Type Coercion**: Explain type coercion in JavaScript with an example.
3. **Object Type Check**: How do you check if a data type is an object type?
4. **Null vs Undefined**: Explain the difference between `null` and `undefined`.
5. **Function Declaration**: What are the different ways to declare a function in JavaScript?
6. **Callback Function**: Explain the purpose of a callback function.
7. **Array Methods**: What is the difference between `map`, `filter`, and `reduce`?
8. **Spread Operator**: What is the spread operator and when is it useful?
9. **Hoisting**: Explain Hoisting in JavaScript.
10. **JSON Stringify**: How do you stringify a JavaScript object into a JSON string?
11. **Array Includes**: How can you check if an array includes a specific element?
12. **Function Definition**: What is a JavaScript function?
