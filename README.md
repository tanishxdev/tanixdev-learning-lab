## Learning Philosophy

### Structured Progression

- **Topic-first approach**: Each folder represents a coherent domain
- **Depth over breadth**: Extensive problem sets per concept (50+ questions per topic)
- **Theory + Practice**: Every concept paired with implementation
- **Iterative improvement**: Code evolves as understanding deepens

### Documentation-Driven

- Markdown files serve as learning journals
- Detailed notes accompany every implementation
- Interview Q&A organized by difficulty and frequency
