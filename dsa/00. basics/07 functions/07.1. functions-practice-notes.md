## Solution 1: Square of a Number

### Thought Process

* We need a function
* Input â†’ one number
* Output â†’ square of that number
* Square means `x * x`

### Code

```cpp

}
```


## Solution 2: Swap Two Numbers (Pass by Reference)

### Thought Process

* Swapping requires changing original values
* Pass by value will fail
* So we must use **reference (&)**

### Code

```cpp
void swapNumbers(int &a, int &b)
{
    int temp = a;
    a = b;
    b = temp;
}
```

Usage:

```cpp
int x = 10, y = 20;
swapNumbers(x, y);
// x = 20, y = 10
```

## Solution 4: Maximum of Two Numbers

### Thought Process

* Compare two values
* Return the larger one
* Simple `if-else`

### Code

```cpp
int findMax(int a, int b)
{
    if (a > b)
        return a;
    else
        return b;
}
```

Usage:

```cpp
int m = findMax(10, 20);   // m = 20
```

## Solution 5: Pass by Value (No Change in Main)

### Thought Process

* Copy of variable is passed
* Change happens only inside function

### Code

```cpp
void changeValue(int x)
{
    x = 100;   // changes only local copy
}
```

Usage:

```cpp
int a = 10;
changeValue(a);
cout << a;   // Output: 10
```

### Why?

* `x` is a copy
* `a` remains unchanged

---

## Solution 6: Pass by Reference (Change Reflects)

### Thought Process

* Reference links to original variable
* Any change affects original

### Code

```cpp
void changeValue(int &x)
{
    x = 100;   // modifies original variable
}
```

Usage:

```cpp
int a = 10;
changeValue(a);
cout << a;   // Output: 100
```

---

## Key Patterns to Remember

| Situation                    | Use               |
| ---------------------------- | ----------------- |
| No modification needed       | Pass by value     |
| Modification required        | Pass by reference |
| Large data (arrays, vectors) | Pass by reference |
| Safety more important        | Pass by value     |
