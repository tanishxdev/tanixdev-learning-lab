# PART A – ARRAYS

## 1. What is an Array?

Array = Collection of same type elements stored in **continuous memory**.

Example:

```
int arr[5] = {10, 20, 30, 40, 50};
```

Memory view:

```
Index:  0   1   2   3   4
Value: 10  20  30  40  50
```

---

## 2. Why Array is Needed?

Instead of:

```
int a1, a2, a3, a4, a5;
```

We use:

```
int arr[5];
```

Benefits:
• Easy iteration
• Better organization
• Efficient memory handling

---

## 3. Array Declaration

Syntax:

```
dataType arrayName[size];
```

Examples:

```
int arr[5];
float prices[10];
char letters[26];
```

Initialization:

```
int arr[5] = {1,2,3,4,5};
```

---

## 4. Accessing Elements

Using index (zero based):

```
arr[0] = first element
arr[4] = last element
```

Example:

```
cout << arr[2];  // prints 3rd element
```

---

## 5. Iterating an Array

Using for loop:

```cpp
for(int i = 0; i < 5; i++) {
    cout << arr[i] << " ";
}
```

---

## 6. Taking Array Input

```cpp
int arr[5];

for(int i = 0; i < 5; i++) {
    cin >> arr[i];
}
```

---

## 7. Common Mistakes

1. Accessing out of bound index

```
arr[5]   // invalid (last index is 4)
```

2. Not initializing array
3. Confusing index with position

# PART B – STRINGS

## 1. What is a String?

String = sequence of characters.

In C++:

```
string name = "Tanish";
```

---

## 2. Taking String Input

### Method 1 – Single word

```
string name;
cin >> name;
```

Stops at space.

### Method 2 – Full line

```
getline(cin, name);
```

---

## 3. String Internally

```
"Hello"
Indices:
H e l l o
0 1 2 3 4
```

---

## 4. Common String Operations

### Length

```
name.length()
```

### Access character

```
name[0]
```

### Modify

```
name[0] = 'A';
```

---

## 5. Example Program (Array + String)

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5];
    string name;

    cout << "Enter 5 numbers: ";
    for(int i = 0; i < 5; i++)
        cin >> arr[i];

    cout << "Enter your name: ";
    cin >> name;

    cout << "Array: ";
    for(int i = 0; i < 5; i++)
        cout << arr[i] << " ";

    cout << endl;
    cout << "Name: " << name;
    return 0;
}
```

---

## 6. Difference: Array vs String

| Array                  | String                 |
| ---------------------- | ---------------------- |
| Stores numbers/objects | Stores characters      |
| Fixed size             | Flexible (std::string) |
| Uses index             | Uses index             |

---

## 7. Dry Run Example

```
int arr[3] = {5, 10, 15};
cout << arr[1];
```

Output:

```
10
```

## Mini Practice

### 1. Sum of 5 Numbers

**Problem Statement**
Input **5 integers** from the user and print their **sum**.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int sum = 0;
    int x;

    // Loop runs exactly 5 times
    for (int i = 1; i <= 5; i++)
    {
        cin >> x;      // take input
        sum += x;      // add to sum
    }

    cout << "Sum = " << sum;
    return 0;
}
```

---

### 2. Length of a String

**Problem Statement**
Input a **string** and print the **number of characters** in it.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    string s;
    cin >> s;   // takes input without spaces

    int length = 0;

    // Count characters one by one
    for (char c : s)
    {
        length++;
    }

    cout << "Length = " << length;
    return 0;
}
```

### 3. Reverse an Array Manually

**Problem Statement**
Input an array and reverse it **without using any built-in function**.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n;
    cin >> n;

    int arr[n];

    // Input array elements
    for (int i = 0; i < n; i++)
    {
        cin >> arr[i];
    }

    int start = 0;
    int end = n - 1;

    // Swap elements from both ends
    while (start < end)
    {
        swap(arr[start], arr[end]);
        start++;
        end--;
    }

    // Print reversed array
    for (int i = 0; i < n; i++)
    {
        cout << arr[i] << " ";
    }

    return 0;
}
```
