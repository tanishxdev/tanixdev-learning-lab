## A. Array Practice Questions

### 1. Sum of Elements in an Array

**Problem Statement**
You are given an array of integers. Find the **sum of all elements**.

#### Thought Process

* An array stores multiple values together
* To access all values, we must **loop**
* Keep one variable to **accumulate the sum**

#### Step-by-Step Logic

1. Initialize `sum = 0`
2. Start from index `0`
3. Add current element to `sum`
4. Move to next index
5. Stop when index reaches array size

#### Key Idea

> One loop + one variable = aggregation problem

---

### 2. Find Maximum Element in an Array

**Problem Statement**
Find the **largest number** in a given array.

#### Thought Process

* We need to **compare elements**
* Start with first element as maximum
* Update maximum whenever a larger value appears

#### Step-by-Step Logic

1. Assume `max = arr[0]`
2. Loop from index `1`
3. If `arr[i] > max`, update `max`
4. End loop → `max` holds answer

#### Common Mistake

* Initializing `max = 0` (fails for negative arrays)

---

### 3. Count Even and Odd Numbers

**Problem Statement**
Count how many **even** and **odd** numbers are in an array.

#### Thought Process

* Even number → divisible by 2
* Odd number → not divisible by 2
* Maintain two counters

#### Step-by-Step Logic

1. Initialize `even = 0`, `odd = 0`
2. Traverse array
3. Check `arr[i] % 2`
4. Increase respective counter

---

### 4. Reverse an Array (Manual)

**Problem Statement**
Reverse the array **without using extra array**.

#### Thought Process

* First element swaps with last
* Second with second last
* Move inward

#### Step-by-Step Logic

1. Set `start = 0`, `end = n-1`
2. Swap `arr[start]` and `arr[end]`
3. Increment `start`, decrement `end`
4. Stop when `start >= end`

#### Key Insight

> Two-pointer technique

## B. String Practice Questions

### 5. Find Length of a String (Manual)

**Problem Statement**
Find the length of a string **without using `.length()`**.

#### Thought Process

* A string is a sequence of characters
* Count characters one by one

#### Step-by-Step Logic

1. Initialize `count = 0`
2. Traverse each character
3. Increment count
4. End → count is length

---

### 6. Count Vowels in a String

**Problem Statement**
Count how many vowels are present in a string.

#### Thought Process

* Vowels are fixed: `a e i o u`
* Compare each character

#### Step-by-Step Logic

1. Initialize `vowelCount = 0`
2. Traverse string
3. If character is vowel → increment count
4. Print result

#### Edge Case

* Handle both uppercase and lowercase

---

### 7. Reverse a String

**Problem Statement**
Reverse a string manually.

#### Thought Process

* Same logic as array reverse
* Strings are index-based

#### Step-by-Step Logic

1. Start pointer at beginning
2. End pointer at last index
3. Swap characters
4. Move pointers inward

---

### 8. Check Palindrome String

**Problem Statement**
Check if a string reads the same forward and backward.

#### Thought Process

* Compare first and last characters
* If mismatch → not palindrome

#### Step-by-Step Logic

1. Set `left = 0`, `right = n-1`
2. If `s[left] != s[right]` → stop
3. Else move inward
4. If loop completes → palindrome

---

## Patterns You Should Notice

| Pattern            | Used In          |
| ------------------ | ---------------- |
| Single loop        | Sum, count       |
| Comparison         | Max, palindrome  |
| Two pointers       | Reverse          |
| Condition checking | Even/odd, vowels |

#include <bits/stdc++.h>
using namespace std;

/*
==================================================
Problem 1: Sum of Elements in an Array
==================================================
*/
int arraySum(vector<int>& arr)
{
    int sum = 0;

    for (int i = 0; i < arr.size(); i++)
    {
        sum += arr[i];   // add each element
    }

    return sum;
}

/*
==================================================
Problem 2: Find Maximum Element in an Array
==================================================
*/
int findMaximum(vector<int>& arr)
{
    int maxi = arr[0];   // assume first element is maximum

    for (int i = 1; i < arr.size(); i++)
    {
        if (arr[i] > maxi)
            maxi = arr[i];
    }

    return maxi;
}

/*
==================================================
Problem 3: Count Even and Odd Numbers
==================================================
*/
void countEvenOdd(vector<int>& arr, int& even, int& odd)
{
    even = 0;
    odd = 0;

    for (int i = 0; i < arr.size(); i++)
    {
        if (arr[i] % 2 == 0)
            even++;
        else
            odd++;
    }
}

/*
==================================================
Problem 4: Reverse an Array (Manual)
==================================================
*/
void reverseArray(vector<int>& arr)
{
    int start = 0;
    int end = arr.size() - 1;

    while (start < end)
    {
        swap(arr[start], arr[end]);
        start++;
        end--;
    }
}

/*
==================================================
Problem 5: Find Length of String (Manual)
==================================================
*/
int stringLength(string s)
{
    int count = 0;

    for (char c : s)
        count++;

    return count;
}

/*
==================================================
Problem 6: Count Vowels in a String
==================================================
*/
int countVowels(string s)
{
    int count = 0;

    for (char c : s)
    {
        c = tolower(c);  // handle uppercase

        if (c == 'a' || c == 'e' || c == 'i' ||
            c == 'o' || c == 'u')
        {
            count++;
        }
    }

    return count;
}

/*
==================================================
Problem 7: Reverse a String
==================================================
*/
string reverseString(string s)
{
    int left = 0;
    int right = s.length() - 1;

    while (left < right)
    {
        swap(s[left], s[right]);
        left++;
        right--;
    }

    return s;
}

/*
==================================================
Problem 8: Check Palindrome String
==================================================
*/
bool isPalindrome(string s)
{
    int left = 0;
    int right = s.length() - 1;

    while (left < right)
    {
        if (s[left] != s[right])
            return false;

        left++;
        right--;
    }

    return true;
}

/*
==================================================
Main Function (Testing All)
==================================================
*/
int main()
{
    vector<int> arr = {1, 2, 3, 4, 5};

    cout << "Array Sum: " << arraySum(arr) << endl;
    cout << "Maximum Element: " << findMaximum(arr) << endl;

    int even, odd;
    countEvenOdd(arr, even, odd);
    cout << "Even Count: " << even << ", Odd Count: " << odd << endl;

    reverseArray(arr);
    cout << "Reversed Array: ";
    for (int x : arr) cout << x << " ";
    cout << endl;

    string s = "level";

    cout << "String Length: " << stringLength(s) << endl;
    cout << "Vowel Count: " << countVowels(s) << endl;
    cout << "Reversed String: " << reverseString(s) << endl;
    cout << "Is Palindrome: " << (isPalindrome(s) ? "Yes" : "No") << endl;

    return 0;
}
