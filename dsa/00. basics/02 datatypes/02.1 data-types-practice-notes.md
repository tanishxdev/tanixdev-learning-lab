# SOLUTIONS (WITH THINKING PROCESS)

## Q1 Solution: Declare Basic Variables

### Thinking

* Age â†’ whole number â†’ `int`
* Height â†’ decimal â†’ `float` or `double`
* First letter â†’ single character â†’ `char`

### Code

```cpp

float height = 5.6;
char firstLetter = 'A';

cout << age << endl;
cout << height << endl;
cout << firstLetter << endl;
```

## Q2 Solution: Integer vs Decimal

### Thinking

* `int` cannot store decimals
* Decimal part is discarded

### Code

```cpp
int x = 3.9;
cout << x;
```

### Output

```
3
```

---

## Q3 Solution: Character Storage

### Thinking

* `char` stores exactly one character
* Must use single quotes

### Code

```cpp
char grade = 'A';
cout << grade;
```

---

## Q4 Solution: Boolean Variable

### Thinking

* `bool` stores `true` or `false`
* Printed as `1` or `0`

### Code

```cpp
bool isPassed = true;
cout << isPassed;
```

### Output

```
1
```

## Q5 Solution: Size of Data Types

### Thinking

* `sizeof()` gives memory in bytes

### Code

```cpp

cout << sizeof(char) << endl;
cout << sizeof(bool) << endl;
```

## Q7 Solution: ASCII Value Behavior

### Thinking

* `char` internally stores ASCII values

### Code

```cpp

```

### Output


## Q8 Solution: Implicit Type Conversion

### Thinking

* `int` is automatically converted to `double`
* Result is `double`

### Code

```cpp
int a = 5;
double b = 2.5;
cout << a + b;
```

### Output

```
7.5
```

---

## Q9 Solution: Explicit Type Conversion

### Thinking

* Manual type casting forces conversion
* Decimal part is lost

### Code

```cpp
double x = 9.8;
int y = (int)x;

cout << y;
```

### Output

```
9
```

---

## Q10 Solution: Predict the Output

### Code

```cpp
int a = 10;
double b = 3.7;
cout << a + b;
```

### Thinking

* `a` becomes `10.0`
* `10.0 + 3.7 = 13.7`

### Output

```
13.7
```
