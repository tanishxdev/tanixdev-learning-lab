# JavaScript Functions — Practice Sheet

## Section 1: Basics (Function Fundamentals)

1. Write a function that prints **“Hello World”**.
2. Write a function that takes a name and prints **“Hello, <name>”**.
3. Write a function that returns the **sum of two numbers**.
4. Write a function that checks whether a number is **even or odd**.
5. Write a function that returns the **square of a number**.
6. Write a function that returns the **maximum of two numbers**.
7. Write a function that returns the **minimum of three numbers**.
8. Write a function with **default parameters**.
9. Write a function that accepts no arguments but returns the **current year**.
10. Write a function that returns `true` if input is a string, else `false`.

---

## Section 2: Parameters & Return Values (Very Important)

11. Write a function that calculates **simple interest**.
12. Write a function that converts **Celsius to Fahrenheit**.
13. Write a function that returns the **length of a string** (without `.length`).
14. Write a function that counts **vowels in a string**.
15. Write a function that returns the **sum of elements of an array**.
16. Write a function that returns the **largest element in an array**.
17. Write a function that returns the **smallest element in an array**.
18. Write a function that checks if a string is a **palindrome**.
19. Write a function that checks if a number is **prime**.
20. Write a function that returns **factorial of a number**.

---

## Section 3: Function Expressions & Arrow Functions

21. Convert a normal function into a **function expression**.
22. Convert the same into an **arrow function**.
23. Write an arrow function that returns the **sum of array elements**.
24. Write an arrow function that checks if a number is **positive**.
25. Difference between:

- function declaration
- function expression
- arrow function (theory)

---

## Section 4: Scope & Closures (Interview Favorite)

26. Demonstrate **local vs global scope** using functions.
27. Write a function inside another function and access outer variables.
28. Create a **counter function** using closure.
29. Create a function that remembers how many times it was called.
30. Explain why closures are useful (with example).

---

## Section 5: Callbacks & Higher-Order Functions

31. Write a function that accepts another function as an argument.
32. Create a callback-based calculator (`add`, `subtract`, etc.).
33. Write your own version of `forEach` using functions.
34. Write a function that applies a callback to every array element.
35. Explain what a **higher-order function** is.

---

## Section 6: Arguments, Rest & Spread

36. Write a function that accepts **any number of arguments** and returns their sum.
37. Write a function using **rest parameters**.
38. Use **spread operator** to pass array values into a function.
39. Difference between `arguments` object and rest parameters.
40. Write a function that merges two arrays using spread.

---

## Section 7: Recursion (Must Know)

41. Write a recursive function to print numbers **1 to N**.
42. Write a recursive function to calculate **factorial**.
43. Write a recursive function to **reverse a string**.
44. Write a recursive function to find **sum of array elements**.
45. Explain recursion vs loops (pros & cons).

---

## Section 8: Real-World Style Problems

46. Write a reusable **validation function** (email / password).
47. Write a function to **debounce** a button click (logic only).
48. Write a function to **throttle** an API call (logic only).
49. Write a function that formats user input (trim, lowercase, etc.).
50. Write a function that returns another function (factory pattern).

51. What is **hoisting** in functions?

53. Why arrow functions don’t have their own `this`?
54. When should you **not** use arrow functions?
55. Pure vs impure functions.
56. What are side effects in functions?
57. Explain function currying with example.
58. Explain IIFE and use-case.
59. Explain call stack using function calls.
