# JavaScript Loops — Practice Sheet

## Section 1: Warm-up (Beginner)

1. Print numbers from **1 to 10** using a `for` loop.
2. Print even numbers between **1 and 50**.
3. Print numbers from **10 to 1** (reverse order).
4. Print the table of **5**.
5. Calculate the **sum of numbers from 1 to N** (N given).
6. Count how many numbers between **1 and 100** are divisible by **3**.
7. Print all characters of a string using a loop.
8. Find the **length of a string without using `.length`**.
9. Print all elements of an array using:
   - `for`
   - `while`

10. Print only **positive numbers** from an array.
