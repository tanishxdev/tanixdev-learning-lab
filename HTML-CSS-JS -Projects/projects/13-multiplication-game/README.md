# Multiplication Game

A simple multiplication game built using pure HTML, CSS, and JavaScript.

## Features

- Random multiplication questions
- Score tracking
- Instant feedback

## Concepts Covered

- DOM manipulation
- Event handling
- Random number generation
- State management

## How It Works

1. Two random numbers are generated
2. User inputs the product
3. Answer is validated
4. Score updates
5. New question loads

## Improvements Possible

- Difficulty levels
- Timer
- Keyboard input support
- Persistent score using localStorage

## Tech Stack

- HTML
- CSS
- JavaScript
