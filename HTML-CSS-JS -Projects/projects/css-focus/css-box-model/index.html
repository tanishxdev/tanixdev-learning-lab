How the Box Model Works

Content

The actual content of the box (text, images, etc.)

Padding

Space between content and border. Affects background.

Border

Edge around padding. Can have style, width, and color.

border: 3px solid #3b82f6;

Margin

Space outside border. Transparent, separates from other elements.
